# mbb-ppt-generator (plugin)

The MBB PPT Generator skill, packaged as a Claude plugin. Install via:

```
claude plugin marketplace add albertojb/mbb-ppt-generator
claude plugin install mbb-ppt-generator
```

Or in Cowork, ask Claude: *"Add the marketplace at albertojb/mbb-ppt-generator and install the mbb-ppt-generator plugin."*
